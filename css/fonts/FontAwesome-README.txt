Font Awesome Icons Setup
========================

To use Font Awesome icons locally, please:

1. Download Font Awesome from: https://fontawesome.com/download
2. Extract the webfonts folder 
3. Copy these files to this directory:
   - fa-solid-900.woff2
   - fa-solid-900.woff

Alternative: Use Font Awesome CDN fallback (requires internet)
Add to your HTML head:
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

The application will work with system fallback icons if Font Awesome is not available.
